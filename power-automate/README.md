## Key Concepts

- **Cloud Flows** — API-first automation. Triggered by events, schedules, or HTTP.
- **Desktop Flows (PAD)** — UI automation for legacy apps with no API.
- **Solutions** — Package flows for proper ALM (Dev → Test → Prod).
- **Connectors** — Standard (free) vs Premium (paid license required).
- **AI Builder** — Microsoft's built-in AI — runs inside the Microsoft boundary (enterprise safe).
